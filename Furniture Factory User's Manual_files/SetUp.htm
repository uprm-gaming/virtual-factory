SetUp

 

The SetUp screen is the first screen that needs to be used when you start a game. If the player do not set up the game though this screen, then it will be impossible to start it. The setup screen requires a configuration step by step. In this window you set the following: Resources, Storages, Unit Load, Reorder Point and Order Quantity for purchases, Workers for each activity, and Execution priority for each activity..

 

 

1.1.1. 

In order to set the areas previously mentioned the player needs to click the icon at the left of each number, which will open a window in order for the player to choose the corresponding settings. If the player doesn’t wants to do settings one by one, the player should click on the DEFAUL SETUP button and every step will be setup automatically. If you want to open the window at some moment just go to the upper right corner of the game and click the SETUP button. That will open the window again for you.
