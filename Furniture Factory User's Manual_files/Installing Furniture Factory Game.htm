Installing Furniture Factory Game

 

In order to install the Furniture Factory Game first you need to go to the UPRM Industrial Engineering web page and download the installer. Once the installer is downloaded proceed to to run it. You will receive a window where you’ll have to click the RUN button in order to allow the program to run. In Linux you should open a Terminal, type “sh” and the full path of the installer location plus the installer name file.

 

 

After running the installer you will see the installer wizard, which will guide you through the install process. Proceed to click the NEXT button in order to continue. In the case you have decided at this point to exit the installation wizard just click the CANCEL button.

 

 

Once you have clicked the NEXT button you will see the game User License Agreement. You must read this window and agree to the terms specified. Click the I ACCEPT THE AGREEMENTbutton, and then proceed to click the NEXT button

 

 

Once the agreement is accepted and you select to continue, you will proceed to select the destination folder for the installation files of the game. You should use the BROWSE button in order to search for the desired folder to use. Once you have selected the installation folder click the NEXT button in order to continue.

 

 

At last you will need to select a Start Menu folder where you want the SetUp to create a program’s shortcut. If you don’t want to create a Start Menu folder just unselect the checkmark button that says CREATE A START MENU FOLDER. After selecting the Start Menu folder click the NEXT button and the installing process will begin.

 

 

Once the installation process has finalized you will receive a window stating that the installation process has completed. Click the FINISH button and proceed to select the installed icons in your computer in order to launch the program
